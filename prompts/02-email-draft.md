# 실습 2 · 미팅 노트로 회신 이메일 초안 만들기

## 시나리오
주간 미팅 직후, 상위자(교수님·박사님·협업자)에게 후속 메일을 보내야 한다. 미팅 노트에서 (1) 결정 사항, (2) 다음 액션, (3) 본인이 해야 할 일을 뽑아 정중한 회신 초안을 만든다.

## 입력
- `meeting-note.md` (이번 repo 동봉 raw 회의록)

## Claude 명령

```
claude
> meeting-note.md 읽고, 미팅에서 결정된 사항과 내가 해야 할 액션을 추려서
> 정중한 톤의 회신 이메일 초안 markdown 으로 작성해줘.
> 수신자는 PI (이혜원 박사님), 발신자는 나(이규민) 가정.
> 다음 단계 일정은 1주 이내 진행 가능한 범위로.
```

또는 더 간단히:
```
> meeting-note.md → 회신 이메일 초안 (markdown, 정중한 톤)
```

## 기대 출력 (예시 구조)
```
제목: [Re] <미팅 주제> 후속 정리 및 다음 단계

박사님 안녕하세요,

오늘 미팅에서 주신 피드백 잘 정리했습니다.
주요 결정 사항은 다음과 같습니다:
- ...
- ...

다음 단계로 1주 내 다음을 진행하겠습니다:
- ...
- ...

추가로 확인이 필요한 부분이 있어 여쭤봅니다:
- ...

감사합니다.
이규민 드림
```

## 핵심 학습 포인트
- Claude 는 raw 회의록 prose 에서 **결정·액션·질문 3 축**을 추출
- 톤(정중/일상/공식) 명시 시 어조 자동 조정
- 수신자·발신자 정보 제공 시 인사·서명까지 자동 처리

## 응용
- 발신 후 답신을 받으면 같은 흐름으로 next-step 메일 자동화 가능
- 주간 보고서·후속 미팅 안건 정리에 그대로 재활용
