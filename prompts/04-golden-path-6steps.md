# 골든패스 6단계 · 첫 세션 실전 흐름 (슬 50)

## 시나리오
즉흥 예시 도중 막혔을 때 즉시 복귀할 수 있는 6단계. claude 첫 실행부터 산출물 생성, CLAUDE.md 업데이트까지 한 루프로 묶는다.

## 6단계 명령

```bash
claude --version
```

```bash
claude  # 실행 + 로그인
```

```
현재 폴더를 읽어줘.
```

```
meeting-note.md를 정리해줘.
```

```
report/report.html을 만들어줘.
```

```
CLAUDE.md에 3줄 규칙을 추가해줘.
```

## 핵심 학습 포인트
- 막혔을 때 이 6단계로 복귀
- 폴더 인식 -> 입력 정리 -> 산출물 생성 -> 규칙 누적 흐름
- 한 세션이 끝나기 전에 CLAUDE.md를 갱신해 다음 세션의 출발점을 높임
