# 실전 스킬 예시 · /clip 만들기와 사용 (슬 73)

## 시나리오
스크린샷을 대화에 붙이는 반복 작업을 명령 하나로 줄인다. 본인 OS와 스크린샷 폴더에 맞춘 `/clip` 스킬을 Claude가 직접 생성하도록 한다.

## 만들기 · Claude에게 줄 프롬프트

```
내 환경에 /clip slash command를 만들어줘.

목표
- 최근 스크린샷 목록 표시
- /clip 1로 1번 이미지 첨부
- /clip 1 질문 으로 이미지+질문 동시 전송
- /clip 1-3 다중 선택

먼저 OS와 스크린샷 폴더를 확인하고
구현 계획을 먼저 보여줘.
```

## 사용 · 만든 뒤의 호출

```
/clip              최근 캡처 목록
/clip 1            1번 이미지 분석
/clip 1 이 에러 고쳐줘
/clip 1-3          여러 장 선택
```

## 쓰임

- 논문 그림 해석
- 에러 화면 디버깅
- 실험 장비 캡처 설명

## 핵심 학습 포인트
- 반복 워크플로우를 스킬로 패키징하면 호출 비용이 한 줄로 줄어듦
- 스킬 생성도 Claude에게 위임 가능. "OS·폴더 확인 후 계획부터"가 안전 가드
- 이미지 첨부와 질문을 한 명령에 합쳐 컨텍스트 손실 없이 다중 캡처 처리
