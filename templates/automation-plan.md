# 내 1페이지 자동화 plan (슬 90)

3분 동안 빈 칸을 채워보세요. 세미나 후 자기 자리로 돌아가 첫 시도 1주 내, 다음 단계 2주 내가 목표.

## 빈 칸

```
① 반복하는 작업: ________________________

② 현재 소요 시간: ________________________

③ Claude Code 로 줄일 단계: ________________________

④ 첫 시도 (1주 내): ________________________

⑤ 다음 단계 (2주 내): ________________________
```

## 작성 가이드

| 항목 | 작성 요령 |
|------|-----------|
| ① 반복하는 작업 | 매일·매주 손으로 하는 일 하나. 예: "주간 보고 작성", "raw NGS 결과 표 정리" |
| ② 현재 소요 시간 | 평균 몇 분, 몇 시간 |
| ③ Claude Code 로 줄일 단계 | 전체 워크플로우 중 어디를 AI가 대신할 수 있는지 |
| ④ 첫 시도 (1주 내) | 가장 작은 시도. 예: "/init 후 CLAUDE.md 작성" |
| ⑤ 다음 단계 (2주 내) | 스킬, 자동화로 굳히기 |

## 핵심 원칙

- 1주는 첫 실험, 2주는 스킬·자동화로 굳히기
- 너무 큰 목표보다 가장 작은 한 단계가 안전
- 자동화 가능 단계와 사람이 마지막 점검할 단계를 분리
